# Frequently Asked Questions (FAQ)

This section answers common user questions.

## ‚ùì General Questions

### What is AI Plugin Builder?

**AI Plugin Builder** is a WordPress plugin that uses artificial intelligence to automatically generate and install WordPress plugins. You simply describe your needs in plain language and the plugin is ready.

### Do I need programming knowledge?

No! You don't need any programming knowledge. Just describe your needs in simple language and the plugin will be ready.

### What types of plugins can I build?

You can build various types of plugins:
- Simple plugins (Shortcode, Widget)
- Plugins with settings pages
- Plugins with Custom Post Type
- Plugins with REST API
- Plugins with database
- And more...

### Are generated plugins secure?

Yes! The plugin undergoes complete security checks before installation:
- SQL Injection check
- XSS check
- CSRF check
- File Inclusion check
- WordPress standards compliance

## üí∞ Cost Questions

### Is the plugin free?

**AI Plugin Builder** plugin is free, but you need an account with the relevant provider to use AI APIs.

### How much does API cost?

API cost depends on the provider:
- OpenAI: Based on usage (Token)
- Anthropic: Based on usage (Token)
- Custom: Depends on provider

### Can I use free API?

Some providers offer limited free API. For more information, visit the provider's website.

## üöÄ Usage Questions

### How do I get started?

1. Install and activate the plugin
2. Enter API Key in settings
3. Go to "Create Plugin" section
4. Describe your needs

For more information, see the [Getting Started](/guide/getting-started) section.

### How can I build better plugins?

To build better plugins:
- Write your request clearly and precisely
- Provide more details
- Use examples
- Break requests into smaller parts

### Can I edit the plugin later?

Yes! You can edit the plugin:
- From the chat interface
- Directly in files
- Request improvements

### How do I delete a plugin?

1. Go to "My Plugins" section
2. Select the desired plugin
3. Click "Delete"
4. Confirm
